Пояснения к конфигурации файла KeyMap для JPEGView

JPEGView позволяет настраивать действия каждой клавиши и комбинаций клавиш.

О переводе

Переводчик: Дмитрий Ерохин

Последнее обновление: 7-FEB-2023

Главное

Чтобы изменить клавиши и действия по своему вкусу, нужно отредактировать файл "KeyMap.txt".

Формат: 

      <Сочетание клавишей или кнопок мыши>			<команда>

ВНИМАНИЕ! Несколько клавишей предопределены и не могут быть изменены:
  • F1 — справка
  • Alt+F4 — выход
  • 0..9 — слайдшоу

Клавиши

Распознаются следующие клавиши:
  • Alt, Ctrl, Shift
  • Esc, Return, Space, End, Home, Back, Tab, PgDn, PgUp
  • Left, Right, Up, Down, Insert, Del, Plus, Minus, Mul, Div, Comma, Period
  • A .. Z
  • F1 .. F12

Мышь

Команды также можно привязывать к кнопкам мыши.
Будьте осторожны, если переопределяете функциональность по умолчанию (например, назначаете панорамирование на левую кнопку мыши)!

Кнопки мыши можно сочетать с клавишами Alt, Ctrl, Shift, но не с другими клавишами или кнопками мыши.

Распознаются следующие кнопки мыши:

  • MouseL, MouseR, MouseM
  • MouseX1, MouseX2
  • MouseDblClk
(левая, правая, средняя, дополнительные кнопки, двойной щелчок левой кнопкой)

Правила

На одну и ту же команду можно назначать несколько клавиш, но нельзя назначать несколько команд на одну клавишу.

ВНИМАНИЕ: Не повредите файл "KeyMap.txt", иначе не будут работать никакие клавиши!
